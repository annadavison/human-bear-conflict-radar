# Model Development Files

This folder contains the scripts used to develop the **conflict risk model** and the **bear dispersal model** alongside some supporting files. Not all data used is included here due to its sensitive nature but these files are described within the script. CORINE land cover 2018 (https://doi.org/10.2909/960998c1-1870-4e82-8051-6485205ebbac) and Copernicus DEM COP-DEM_GLO-30-DGED (https://doi.org/10.5270/ESA-c5d3d65) files used in this study are not included here but can be freely downloaded from Copernicus.
